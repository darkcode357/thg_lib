C:> reg add \\HKLM\Software\Get registry value remotely: C:> reg query \\HKLM\Software\

PS C:> Test-Path "HKCU:\Software\Microsoft\"

C:> robocopy C:\\\ /E

PS C:> Test-Path C:\Scripts\Archive* -include ·PSl, ,VbS
